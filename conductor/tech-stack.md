# Tech Stack - $4 Portfolio

## Core Technologies
- **Jekyll:** Used as the static site generator to transform Markdown files into a functional website.
- **Markdown:** The primary format for managing content, project lists, and professional profiles.
- **GitHub Pages:** The hosting platform used for deployment.

## UI/UX
- **jekyll-theme-minimal:** A clean, minimalist Jekyll theme that prioritizes content readability and navigation.

## DevOps & Infrastructure
- **GitHub Actions:** Automates the build and deployment process to GitHub Pages whenever changes are pushed to the repository.
- **Git:** Version control for managing the project's source code and content.
